# feng3d
基于WebGL的3D渲染引擎

开始尝试？ 在[试验台](http://feng3d.com/examples/index.html)尝试引擎的API，同时提供了一些简单的示例去学习如何使用feng3d引擎。

## 编译项目

    `ctrl+shift+B`

    `node ShaderPack.js`
